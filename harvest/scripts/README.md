# Harvested Scripts

## 1. survey_report_generator.py
**Why it's valuable**: Generates an initial EDA (`frequencies_report.md`) using 100% Native Markdown and Mermaid.js (xychart-beta and pie). Removes the physical dependency on exported `.png` charts from Matplotlib/Seaborn. Includes automatic numerical scale detection and a Pearson correlation matrix.

## 2. final_report_generator.py
**Why it's valuable**: Generates a senior-level analytical report. Automatically runs Mann-Whitney U and Kruskal-Wallis H tests on cross-tabulated variables to prove statistical significance of hypotheses. Generates actionable strategy frameworks (Forças, Gargalos, Oportunidades).
