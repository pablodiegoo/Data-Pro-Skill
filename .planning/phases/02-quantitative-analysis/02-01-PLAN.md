**Execution Steps:**

Step 1 — Inspect the data: Scan all provided data for segment names, variable names, sample size (N), metric values (NPS, CSAT, churn rate, etc.), and any notable patterns (outliers, extreme values, unexpected distributions). Step 3 — Generate questions. For each selected category, craft ONE question that: (a) references CONCRETE data elements (segment names, N values, metric values — never generic placeholders), (b) is provocative — challenges assumptions, not confirms them, (c) cannot be answered with yes/no, (d) would change the analysis approach depending on the answer. - Minimum 3 questions, maximum 5 — no fewer, no more
- No agent loop — clarify is pre-analytical, no statistics involved
- No prose wrapping the numbered list — output IS the list
